# `string` vs `bool`

## `string` to `bool`

**Method**
```go
package strconv

func ParseBool(str string) (bool, error)
```

**Example**
```go
strconv.ParseBool("true") // true
```

## `bool` to `string`

**Method 1**
```go
package strconv

strconv.FormatBool(b bool) string
```

**Example**
```go
strconv.FormatBool(true)  // "true"
```

**Method 2**
```go
package fmt

func Sprintf(format string, a ...interface{}) string
```

**Example**
```go
fmt.Sprintf(true)  // "true"
```
